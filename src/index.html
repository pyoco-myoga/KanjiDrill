# 問題
説明 入力「今日の{てんき|天気}は{はれ|晴れ}です」のとき、
問題「今日のてんきははれです」, 解答「天気, 晴れ」となります。
Downloadsボタンを押すと画像がダウンロードできます。

画像は、問題文:黒、背景:透明、解答:白がデフォルトです。この場合は白の上に置くと、うまく解答が隠れます。
注意 問題が長すぎると写りません(日本語だと最大表示数39文字)。
記号「{|}」は半角を用いてください。
書きの問題の例 今日の{てんき|天気}は{はれ|晴れ}です
読みの問題の例 今日の{天気|てんき}は{晴れ|はれ}です
選択問題の例1 筋肉は{(すべて, なにか)|すべて}を解決する。
選択問題の例2 This is {(a, an)|a} pen.
